# KMTools Skills Collection

A collection of Claude Code skills for CLI tools and development workflows.

## Skills

### 1. llm

Access and interact with Large Language Models from the command line using Simon Willison's llm CLI tool. Supports OpenAI, Anthropic, Gemini, Llama, and dozens of other models via plugins.

**Use for:** Running LLM prompts, managing API keys, installing model plugins, working with embeddings, or extracting structured data.

### 2. parakeet

Convert audio files to text using parakeet-mlx, NVIDIA's Parakeet automatic speech recognition model optimized for Apple's MLX framework.

**Use for:** Transcribing podcasts, interviews, meetings, and other audio content.

### 3. yt-dlp

Download audio and video from thousands of websites using yt-dlp. Feature-rich command-line tool supporting format selection, subtitle extraction, playlist handling, and post-processing.

**Use for:** Downloading media, extracting audio, managing playlists, or troubleshooting downloads.

### 4. just

A handy command runner for saving and running project-specific commands. Features recipe parameters, .env file loading, shell completion, and cross-platform support.

**Use for:** Writing justfiles, understanding just syntax, or setting up project automation.

### 5. jj-vcs

Jujutsu (jj) is a powerful Git-compatible version control system with innovative features like automatic rebasing, working-copy-as-a-commit, operation log with undo, and first-class conflict tracking.

**Use for:** Working with jj commands, understanding jj workflows, or migrating from Git to jj.

### 6. piper

Convert text to speech using Piper, a fast, local, neural text-to-speech system with natural sounding voices.

**Use for:** Converting text files or markdown to audio, batch processing documents, creating accessible content.

### 7. mermaid

Generate diagrams and flowcharts from mermaid definitions using the mermaid-cli (mmdc). Supports 20+ diagram types including flowcharts, sequence diagrams, class diagrams, and more.

**Use for:** Creating diagrams, flowcharts, sequence diagrams, Gantt charts, or converting mermaid code to SVG/PNG/PDF.

### 8. jira-cli

Command-line interface for Jira ticket management. Contains essential setup instructions and non-interactive command patterns.

**Use for:** Managing Jira tickets from the command line, avoiding interactive mode blocking.

### 9. browser-tools

Minimal CDP (Chrome DevTools Protocol) tools for browser automation, web scraping, and collaborative site exploration.

**Use for:** Controlling Chrome programmatically, executing JavaScript in page context, taking screenshots, picking DOM elements.
